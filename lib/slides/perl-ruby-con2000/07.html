クラスの例

  • 人を表すPersonクラス 
    $p = new Person name => 'Tom', age => 20;
$p->speak;	# prints "I am Tom, 20 years old."
  • 実装は... 
    package Person;
sub new {
    my $class = shift;		# クラス名が第1引数
    my $self = { @_ };		# 引数をハッシュで受ける
    bless $self, $class;	# クラスに bless する
}
sub speak {
    my $self = shift;		# オブジェクト自身
    printf 'I am %s, %d years old.', $self->{name}, $self->{age};
}
1;
