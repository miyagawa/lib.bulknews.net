blessとは

$obj = bless { }, "ClassName";
  • Perl の変数をクラス名に結び付ける
  • 1引数の場合, 現在のパッケージに blessされる
  • 継承で混乱しないため, 2引数で呼ぶべき 
package Child;
@ISA = qw(Person);		# 継承
sub new {
    my $class = shift;		# クラス名
    my $self = $class->SUPER::new(@_);	# 親クラスの new
						# $self は Child クラス!
    $self->do_my_own_thing;	# Child クラスのメソッド
    return $self;
}
