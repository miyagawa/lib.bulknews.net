INSTALL

% wget eijiro.gz
% gunzip eijiro.gz
# cp eijiro /usr/local/bin

PREREQUISTICS

以下の Perl モジュールが必要です。CPAN shell からインストールしてください。

  • Jcode
  • LWP::Simple
  • URI::Escape
  • HTML::FormatText
  • HTML::TreeBuilder
  • Term::ReadLine

USAGE

コマンドラインから eijiro 単語名 として実行。和英・英和は自動切換です。引数なしで実行するとインタラクティブモードになります。インタラクティブモードは C-d で抜けます。

検索の結果は HTMLレンダリングされ、$ENV{PAGER} によって表示されます。

% eijiro foobar
% eijiro ほげほげ
% eijiro
eijiro> ほげほげ
eijiro> ^D
