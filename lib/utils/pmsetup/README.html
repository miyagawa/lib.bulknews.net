INSTALL

pmsetup には初期クレジットが設定されていますので、適宜変更してください。 
% wget pmsetup.gz
% gunzip pmsetup.gz
% vi pmsetup
# cp pmsetup /usr/local/bin

USAGE

% pmsetup Foo::Bar
% cd Foo-Bar
% perl Makefile.PL
% make test
% make manifest (MANIFEST の作成)
% make dist     (tarball の作成)
