WHAT'S THIS

prom-wl の生成するログファイルから、POP3 の応答を返します。Win32 端末からASTEC-X や TeraTerm 経由で Wanderlust でメールを読んでいる人におすすめします。Win32 からB's biffなどの biff ツールで、UNIX 側の新着メールをチェックするのに便利です。というか、それ以外には使えません。

INSTALL

% wget prompop-0.xx.tar.gz
% tar xzvf prompop-0.xx.tar.gz
% cd prompop-0.xx
% perl Makefile.PL
% make
# make install

USAGE

tcpserver や inetd から起動してください。引数には procmail ログ、MH ディレクトリ、POP3 認証に使うパスワード（crypt したもの）を渡します。 
% tcpserver 0 8110 /use/local/bin/prompop ~/.procmail/log ~/Mail 2aea9ADdi &
