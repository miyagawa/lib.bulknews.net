foafxom

foafxom is a set of plugins and flavours that enables you to use blosxom as a FOAF management tool.

Download

foafxom-0.01.tar.gz (12524 bytes, Jan 30 21:38 JST)

Sample

See my FOAF using foafxom and samples directory distributed with foafxom.

Install

Edit blosxom script (and you'd like to rename it to 'foaf'), move it to cgi executable directory and just access it from your favourite web browser. Your friends, interests and other profiles can be edited as a blog entry of blosxom. See samples directory for how to manage your data. samples directory also has a suitable 'rdf' flavours to display your blog as FOAF.
