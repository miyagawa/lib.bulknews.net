NAME

Image::HTML - converts any image to HTML table

SYNOPSIS

  use Image::HTML;


  my $image = Image::HTML->new;
  $image->read($file);
  my $html = $image->html;


  # functional
  use Image::HTML qw(image2html);
  my $html = image2html('foo.gif');

DESCRIPTION

No images for web pages! Image::HTML converts ANY images to HTML table.
